# Site acadêmico — estilo revista/ensaio

Esta é uma versão inteiramente nova, pensada como um periódico de filosofia pessoal:
- abertura centralizada de autor, como um artigo;
- navegação mínima;
- publicação em destaque;
- cartões de pesquisa tratados como ensaios;
- lista de publicações no formato de catálogo;
- pouca cor e acentos editoriais (verde, ferrugem, azul e areia).

Abra `index.html` no navegador ou publique toda a pasta em GitHub Pages, Netlify ou Vercel.
