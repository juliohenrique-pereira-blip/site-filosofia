## Publicação no GitHub Pages
1. Abra o repositório `site-filosofia` no GitHub.
2. Escolha **Add file → Upload files**.
3. Envie **o conteúdo** desta pasta, preservando a pasta `assets`.
4. Confirme em **Commit changes**.
5. Aguarde alguns minutos e recarregue o site.
